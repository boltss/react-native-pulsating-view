# Usage
```
<PulsatingView animate>
    <Text>This text pulsates</Text>
</PulsatingView>
```

# Props
| Prop     | Type    | Required | Default |
|----------|---------|----------|---------|
| animate  | boolean | no       | false   |
| minScale | number  | no       | 0.95    |
| maxScale | number  | no       | 1       |
